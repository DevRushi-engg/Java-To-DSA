Variables in Java
=================

Variables are used to store data in a program. 
In Java, you must declare a variable before you can use it. 
A variable declaration includes the variable's type and name.

For example, to declare an integer variable named "age", you would write:

```java
int age;
```
In this example, "int" is the data type that specifies that the variable will store integer values, and "age" is the name of the variable.

You can also initialize a variable at the time of declaration. For example:

```java 
int age = 25;
```
In this example, the variable "age" is declared and initialized with the value 25.
You can also declare multiple variables of the same type in a single line. For example:

```java 
int x = 10, y = 20, z = 30;
```

In this example, three integer variables "x", "y", and "z" are declared and initialized with the values 10, 20, and 30 respectively.
