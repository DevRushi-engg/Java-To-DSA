Print Statements
================

Print Statements are outputs informaton to the screen.

Breaking down the print statement:- 
-----------------------------------

System.out.println("Hello World");

1. System: is the built in java class that contains useful tools and methods for program.

2. out: is the short for output. Represents the output stream.

3. println: is the method that prints the output to the console and moves the cursor to the next line.

4. "Hello World": is the string that we want to print. It is enclosed in double quotes.

5. ; (semicolon): is used to terminate the statement in Java.

When we run the above code, it will output "Hello World" to the console.

There are two types of print statements in Java:

1. System.out.println(): This method prints the output and moves the cursor to the next line.

2. System.out.print(): This method prints the output but does not move the cursor to the next line.
